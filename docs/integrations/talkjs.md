# TalkJS

## Variáveis
```
NEXT_PUBLIC_TALKJS_APP_ID=
TALKJS_SECRET_KEY=
```

## Sem chave
- Página de mensagens mostra aviso (não quebra)
- Testes foundation TalkJS exigem chaves válidas

## Ativação
1. App TalkJS
2. Preencher App ID + Secret
3. Rebuild (App ID é `NEXT_PUBLIC_*`)
4. Smoke + `npm run test:foundation:talkjs`
