# Gestão de Custos

`getEnterpriseCostDashboard()` agrega:

- Consumo diário / mensal (tokens + USD estimado)
- Por módulo / role / usuário
- Por ferramenta (`AIToolExecution`)
- Orçamentos `AI_DAILY_BUDGET_USD` / `AI_MONTHLY_BUDGET_USD`
- Alertas 80%/100%

Admin: `/admin/ai/enterprise` (+ página de custos legado).
